# 🎓 Student Skill Card Dashboard

A full-stack web application built with **Flask, HTML, CSS, and JavaScript** that allows teachers to evaluate students’ skills and generate a digital **Skill Card**.  
The system provides role-based dashboards for **teachers** and **students**, with modern UI elements like **glassmorphism**, **charts**, and **performance tracking**.

---

## 🚀 Features
- **Role-based login & registration** (Teacher / Student)
- **Teacher Dashboard**: Add, update, and track student skills
- **Student Dashboard**: View personalized skill card and performance history
- **Interactive Charts**: Visualize academic and skill progress with bar graphs & pie charts
- **Modern UI**: Glassmorphism, gradient backgrounds, and responsive design
- **Database Integration**: Store student records and skill evaluations

---

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Flask (Python)
- **Database**: SQLite / MySQL
- **Version Control**: Git & GitHub
